Pour commencer

J'ai fait ce template dans l'objectif de commencer un sites web avec mes classes CSS de bases et de réduire le temps de à produire un site web. Je planifi de faire une version avec SASS et peut-être en faire un thème WordPress. Ce template vise à maximiser la performance du site et que les configs de base comme l'accessibilité, le PWA et les libraries JS en toute simplicité.
